大学生与家长直连的家教服务平台产品原型

项目简介
校园家教助手是一个面向下沉市场（如保定市）的大学生家教对接微信小程序产品方案。基于个人家教实践经历中发现的"家校不统一"痛点，独立完成从需求分析、竞品对比、功能设计到原型输出的全流程产品设计。

抽成	重度抽成	高额信息费	≤10%
试课机制	无	无	前3节双倍+免费换师

安全保障	平台背书	基本没有	实名认证+实时定位

核心功能亮点
分级考核机制
小学：基础知识 + 交流能力 + 性格匹配
初中：学科知识 + 教学质量（参考教资面试）
高中：分学科深度考核
985/211：免知识考核，只做人格+交流测试
前三节试课双倍机制
降低家长试错成本，激励教师认真备课，提升成交转化率。

AI辅助教学（基于DeepSeek）
生成备课教案和练习题
分析学生薄弱知识点
推荐趣味教学方法
安全机制
实时定位追踪
一键报警
首次上门行为规范
教学状态确认 + 周期递减
冷启动策略
阶段	策略	目标
第一阶段	保定学院校园内测，微信群跑通匹配	10-20大学生 + 5-10家长
第二阶段	校园公众号 + 口碑传播	校园扩散
第三阶段	扩展至河北大学等高校，小程序产品化	城市级别
