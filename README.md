## 🎯 项目简介

校园家教助手是一个基于地理位置的大学生与家长直连家教平台，主打**比机构便宜，比中介靠谱，不满意能换**。

- **需求来源**：基于个人家教实践中的"家校不统一"痛点
- **产品形态**：微信小程序
- **目标用户**：三四线城市有K12辅导需求的家长 + 在校大学生
- **当前阶段**：产品设计与原型验证

## ✨ 核心功能

### 三端用户设计
| 角色 | 核心功能 |
|------|---------|
| 👨‍👩‍👧 **家长端** | 搜索教师、筛选匹配、查看评价、预约试课 |
| 🎓 **学生端** | 收入管理、接单状态、资质考核、AI教学助手 |
| 🏫 **教师端** | 教学经验展示、长期聘用 |

### 差异化机制
- **分级考核**：小学/初中/高中各一套考核题，985/211免知识考核
- **试课激励**：前3节课教师获双倍课时费，不满意免费换师
- **安全体系**：实时定位、一键报警、首次上门行为规范
- **AI辅助教学**：基于大模型的备课教案生成、出题、薄弱点分析
- **工资日结**：平台抽成≤10%，其余直接到账

### 商业模式
| 收入来源 | 说明 |
|---------|------|
| 课时费抽成 | ≤10%，前期低抽成吸引供给 |
| Pro增值服务 | 985/211优先匹配、AI精准匹配 |
| 教学类广告 | 软性植入 |
